QUICK START COMMANDS
====================

1. Install dependencies:
   $ pip install -r requirements.txt

2. Configure environment:
   $ cp .env.example .env
   $ python -c "import secrets; print('SECRET_KEY=' + secrets.token_hex(32))" >> .env

3. Run application:
   $ python run.py

4. Open browser:
   http://localhost:5000
